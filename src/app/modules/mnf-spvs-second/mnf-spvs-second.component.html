คำอธิบาย/หลักฐานอ้างอิง

ใช่

ไม่ใช่

 1.  มีการระบุรหัสวิชา ชื่อรายวิชา กลุ่มสาระการเรียนรู้ ระดับชั้น จำนวนชั่วโมงและ/หรือหน่วยกิตอย่างครบถ้วน ชัดเจน

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

 2.  ชื่อหน่วยการเรียนรู้ น่าสนใจ กะทัดรัด ชัดเจน สอดคล้องกับสาระสำคัญ/ความคิดรวบยอด

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

3.  การจัดกลุ่มตัวชี้วัด/ผลการเรียนรู้ ในแต่ละหน่วยการเรียนรู้มีความสัมพันธ์เชื่อมโยงและสามารถจัดการเรียนรู้ร่วมกันได้

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

4.  สาระสำคัญ/ความคิดรวบยอด แสดงให้เห็นแก่นความรู้รวมถึงสาระการเรียนรู้ของตัวชี้วัด/ผลการเรียนรู้ในแต่ละหน่วยการเรียนรู้

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

5.  เวลาเรียน ในแต่ละหน่วยการเรียนรู้เหมาะสมกับจำนวนตัวชี้วัด/ ผลการเรียนรู้ ความยากง่ายของสาระสำคัญ และกระบวนการเรียนรู้ที่จะใช้ในการพัฒนาผู้เรียนตามตัวชี้วัด/ผลการเรียนรู้

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

6.  น้ำหนัก/สัดส่วนของคะแนน  เหมาะสมกับเวลาเรียน ความยากง่ายของสาระสำคัญ และกระบวนการเรียนรู้ที่จะใช้ในการพัฒนาผู้เรียนตามตัวชี้วัด/ผลการเรียนรู้

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

7.  ชิ้นงาน/ภาระงานรวบยอด สอดคล้อง สัมพันธ์ครอบคลุมและสะท้อนให้เห็นผลการพัฒนาผู้เรียนตามตัวชี้วัด/ผลการเรียนรู้ใน แต่ละหน่วยการเรียนรู้

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

ใช่

ไม่ใช่

 1.  มีการระบุรหัสวิชา ชื่อรายวิชา กลุ่มสาระการเรียนรู้ ระดับชั้น จำนวนชั่วโมงและชื่อหน่วยเรียนรู้อย่างครบถ้วน ชัดเจนและสอดคล้องกับโครงสร้างรายวิชา

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

 2.  มาตรฐานการเรียนรู้ / ตัวชี้วัด / ผลการเรียนรู้ สมรรถนะสำคัญและคุณลักษณะอันพึงประสงค์มีความสัมพันธ์ เชื่อมโยงกัน 

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

3.  สาระสำคัญ / ความคิดรวบยอด มีความถูกต้อง สอดคล้อง และครอบคลุมมาตรฐาน/ตัวชี้วัด/ผลการเรียนรู้  

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

4.  สาระการเรียนรู้  มีความถูกต้อง และสอดคล้องกับมาตรฐาน /ตัวชี้วัด / ผลการเรียนรู้  

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

5.  หลักฐานการเรียนรู้ (ชิ้นงาน / ภาระงาน) สอดคล้องกับมาตรฐานการเรียนรู้ / ตัวชี้วัด / ผลการเรียนรู้ การวัดและประเมินผล     และกิจกรรมการเรียนรู้ 

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

6.  การวัดและประเมินผล 

      -  เป็นการประเมินตามสภาพจริงใช้วิธีการที่หลากหลาย สอดคล้องกับมาตรฐานการเรียนรู้ / ตัวชี้วัด/ผลการเรียนรู้ และกิจกรรมการเรียนรู้

       -  ระบุวิธีการ เครื่องมือและเกณฑ์การประเมินที่สะท้อน

คุณภาพผู้เรียนตามมาตรฐานการเรียนรู้ / ตัวชี้วัด/ผลการเรียนรู้

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

7.  กิจกรรมการเรียนรู้ 

        -  ใช้รูปแบบ / วิธีการ / เทคนิค / ทักษะกระบวนการตามธรรมชาติของวิชา ที่ช่วยพัฒนาผู้เรียนให้บรรลุมาตรฐาน  /ตัวชี้วัด / ผลการเรียนรู้ สมรรถนะสำคัญของผู้เรียนและคุณลักษณะอันพึงประสงค์ได้อย่างเหมาะสมและมีประสิทธิภาพ

       -  สามารถนำพาให้ผู้เรียนสร้างสรรค์ชิ้นงาน หรือภาระงานตามที่กำหนด

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")

8.  สื่อและแหล่งเรียนรู้  มีความเหมาะสมกับกิจกรรมการเรียนรู้ของผู้เรียน และเวลา ช่วยให้ผู้เรียนเกิดการเรียนรู้ได้ดีและรวดเร็วขึ้น

คำอธิบาย/หลักฐานอ้างอิง (ไม่มีให้ใส่ "-")
